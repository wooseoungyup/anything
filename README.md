## 1. 마크다운 이란   
마크다운은 텍스트기반의 마크업언어입니다. 쉽게 쓰고 읽을 수 있으며,          
간단히 기록하고

***
## 2. 마크다운 사용법(문법)
### 1. 헤더

헤더의 크기를 # 을 통해서 조절 할 수 있습니다. 
샵(#) 의 개수에 따라 글자의 크기가 정해지며, 샵(#)이 많을 수록 글자가 작아집니다.
아래는 예시 입니다.

```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6

```
